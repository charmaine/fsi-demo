# Anthropic Brand Guidelines Skill

This skill ensures all generated documents and applications adhere to Anthropic's brand standards for consistent, professional, and human-centered design.

## Brand Identity

### Company: Anthropic
**Tagline**: "AI Safety and Research"
**Core Values**: Human-centered, warm, trustworthy, minimal, sophisticated

## Visual Standards

### Color Palette

**Primary Colors**:
- **Anthropic Tan**: #D4A574 (RGB: 212, 165, 116) - Primary brand color, warm cream/tan
- **Deep Charcoal**: #1a1a1a (RGB: 26, 26, 26) - Primary text, headers
- **Warm White**: #FAF7F2 (RGB: 250, 247, 242) - Backgrounds

**Secondary Colors**:
- **Terracotta**: #CC785C (RGB: 204, 120, 92) - Accents, CTAs
- **Soft Sand**: #E8DFD5 (RGB: 232, 223, 213) - Secondary backgrounds, cards
- **Muted Sage**: #A8B5A0 (RGB: 168, 181, 160) - Success states
- **Dusty Rose**: #C4A4A4 (RGB: 196, 164, 164) - Subtle highlights

**Functional Colors**:
- **Success**: #4A7C59 (RGB: 74, 124, 89) - Positive values, gains
- **Warning**: #D4A574 (RGB: 212, 165, 116) - Cautions (uses primary tan)
- **Error**: #B85450 (RGB: 184, 84, 80) - Negative values, losses
- **Neutral**: #6B6B6B (RGB: 107, 107, 107) - Secondary text

### Typography

**Primary Font Family**: 'Styrene A', 'Inter', system-ui, -apple-system, sans-serif

**Font Hierarchy**:
- **H1**: 32pt, Medium weight, Deep Charcoal
- **H2**: 24pt, Medium weight, Deep Charcoal
- **H3**: 18pt, Medium weight, Deep Charcoal
- **Body**: 14pt, Regular weight, Deep Charcoal
- **Caption**: 12pt, Regular weight, Neutral

### Design Principles

1. **Warmth**: Use warm tones over cold blues/grays. The palette should feel human and approachable.
2. **Minimalism**: Embrace whitespace. Less is more. Remove unnecessary visual elements.
3. **Sophistication**: Clean lines, subtle shadows, refined details.
4. **Accessibility**: Maintain proper contrast ratios. All text must be readable.
5. **Consistency**: Use the color palette exclusively. No off-brand colors.

## Application Standards

### Web Applications

**Layout Rules**:
- Background: Warm White (#FAF7F2)
- Cards/Panels: White (#FFFFFF) with subtle shadow
- Navigation: Deep Charcoal (#1a1a1a) or Warm White
- Borders: Soft Sand (#E8DFD5) or transparent
- Border radius: 8-12px for cards, 4-6px for buttons

**Interactive Elements**:
- Primary buttons: Terracotta (#CC785C) background, white text
- Secondary buttons: Transparent with Deep Charcoal border
- Hover states: Slightly darker shade (10% darker)
- Focus states: 2px Anthropic Tan outline

**Charts & Data Visualization**:
- Primary series: Terracotta (#CC785C)
- Secondary series: Anthropic Tan (#D4A574)
- Tertiary series: Muted Sage (#A8B5A0)
- Positive values: Success green (#4A7C59)
- Negative values: Error (#B85450)
- Grid lines: Soft Sand (#E8DFD5)
- Background: Warm White (#FAF7F2)

### Data Presentation

**Numbers**:
- Use comma separators for thousands
- Currency: $X,XXX.XX format
- Percentages: XX.X% (one decimal)
- Positive changes: Success green with + prefix
- Negative changes: Error red with - prefix

**Tables**:
- Headers: Deep Charcoal text, Soft Sand background
- Alternating rows: White and Warm White
- Right-align numbers
- Left-align text
- Subtle borders in Soft Sand

## Quality Checklist

Before finalizing any design work:
1. ✓ All colors match the Anthropic palette exactly
2. ✓ Typography uses correct font families and weights
3. ✓ Sufficient contrast for accessibility
4. ✓ Consistent spacing and alignment
5. ✓ Warm, human, approachable feel
6. ✓ Minimal and uncluttered

## Prohibited Elements

Never use:
- Cold blues or corporate navy colors
- Gradients (unless very subtle)
- Heavy drop shadows
- Decorative fonts
- Cluttered layouts
- Generic stock imagery

## Application Instructions

When applying brand guidelines:
1. Start with Warm White background
2. Use Deep Charcoal for primary text
3. Apply Terracotta for primary actions/accents
4. Use Anthropic Tan for secondary highlights
5. Ensure all charts use brand colors
6. Maintain generous whitespace
7. Keep the overall feel warm and human
